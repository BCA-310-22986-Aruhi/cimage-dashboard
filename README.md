Academic Insight Platform – Student Performance Dashboard

This project is a web-based Student Performance Dashboard developed using Python, Streamlit, Pandas, Plotly, Microsoft SQL Server (MSSQL), and Azure SQL Database.

Features

- Secure Admin and Student Login
- Student Performance Analysis
- Attendance Management System
- Department-wise Performance Reports
- Interactive Data Visualizations
- Student and Teacher Profile Management
- Report Generation and Data Filtering
- Role-Based Access Control

Technologies Used

- Python
- Streamlit
- Pandas
- Plotly
- MSSQL
- Azure SQL Database

Project Modules

- Login Authentication
- Dashboard Implementation
- Student Profile Management
- Teacher Profile Management
- Attendance Management
- Report Generation
- Notifications and Settings
- Data Analysis and Visualization

Developed For

Final Year BCA Major Project – CIMAGE, Patna.
